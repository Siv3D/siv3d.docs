## 動作環境

|| IDE での開発 | 実行 | サンプルを URL から開く |
|:--:|:--:|:--:|:--:|
| Chrome | ✅ | ✅*1 | ✅ |
| FireFox | ✅ | ✅*2 | ✅ |
| Edge | ✅ | ✅*1 | ✅ |
| Safari | ✅ | ✅ | |

\*1: シークレットモードまたはサードパーティ Cookie が無効のときは利用できません  
\*2: プライベートブラウジングでは利用できません
