# 29. クッキークリッカー
チュートリアル 3 ～ 28 の内容を使って、クッキークリッカー風のゲームを作ります。

!!! info "クッキークリッカーとは"
    - クッキークリッカーは、クッキーをクリックしてクッキーの数を増やしていくゲームです
    - 増やしたクッキーを使って、クッキーを増やすための生産設備を購入できます
    - 2013 年にオリジナルのゲームが公開されて人気を博し、その後、様々な派生ゲームが作られています
