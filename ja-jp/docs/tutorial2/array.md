# 21. 配列
動的配列クラス `Array` の基本的な使い方を学びます。

## 21.1 Array
- Siv3D では `std::vector<Type>` の代わりに `Array<Type>` で動的配列を扱います
- `Array` は内部で `std::vector` を使用していて、`std::vector` と同様の機能を提供するほか、追加のさまざまなメンバ関数を持ちます
- `std::vector` と同様に、要素がメモリ上に連続で並んでいることが保証されています
