# Siv3D Palette Browser

## Siv3D Palette Browser の機能
Siv3D で色を表現する際に使う `Palette`, `Color`, `ColorF`, `HSV` をブラウザ上で確認し、相互に変換したり、クリップボードにコピーしたりできるツールです。
