# 協賛企業の募集について

### 企業の皆様へ
Siv3D は C++ によるゲーム・アプリケーション開発を楽しく簡単にするツールと技法をすべての人々と共有することを目標に、オープンソースのプログラミングフレームワークの開発・提供、ユーザコミュニティ支援、国内各地での勉強会開催、若い世代のオープンソース参加支援に長年取り組んでいます。Siv3D の趣旨にご賛同のうえ、ご支援・ご協賛を賜りますようお願い申し上げます。
